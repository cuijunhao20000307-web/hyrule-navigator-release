# 海拉鲁全图导航 1.0.4 使用说明

这是《塞尔达传说：旷野之息》1.6.0 的非官方局域网地图伴侣。Switch 上的 Tesla 插件只读获取林克位置和朝向，电脑负责运行地图网站，手机或平板通过同一 Wi-Fi 打开网站。

## 一、适用版本

- 游戏：旷野之息 1.6.0
- Title ID：`01007EF00011E000`
- Build ID：`8E9978D50BDD20B4C8395A106C27FFDE`
- Switch 插件：Hyrule Bridge Ultra 0.5.7
- Switch 服务端口：`37890`
- 网页端口：`3000`

不包含游戏、系统文件、密钥或固件。请勿在不兼容的游戏版本上使用。

## 二、安装 Switch 端

1. 完全关闭 Switch，取出 SD 卡并连接电脑。
2. 打开发布包中的 `Switch端/HyruleBridgeUltra-0.5.7-AutoStop-SD.zip`。
3. 将压缩包中的 `switch` 文件夹复制到 SD 卡根目录并选择合并。
4. 最终文件应位于：`SD:/switch/.overlays/HyruleBridgeUltra.ovl`。
5. 安全弹出 SD 卡，插回 Switch 后正常开机。

不要安装额外 sysmodule。若以前装过旧版 Hyrule Bridge Ultra，请只保留一个同名 `.ovl` 文件。

## 三、启动电脑端

### macOS

1. 安装 Node.js 22 或更高版本。
2. 打开 `电脑端` 文件夹，双击 `启动电脑端.command`。
3. 第一次启动会自动安装依赖，需要联网并等待完成。
4. 启动窗口会显示电脑地址和手机地址。

如果系统阻止脚本运行，可在终端进入 `电脑端` 文件夹后执行：

```sh
npm install
npm run dev -- --host 0.0.0.0
```

### Windows

1. 安装 Node.js 22 或更高版本。
2. 双击 `电脑端/启动电脑端-Windows.bat`。
3. 第一次启动会自动安装依赖。
4. 电脑浏览器打开 `http://localhost:3000`。

## 四、电脑和手机网址

- 当前电脑自己打开：`http://localhost:3000`
- 手机或平板打开：`http://运行网站的电脑IP:3000`

每位使用者的电脑 IP 都可能不同。例如某台电脑的 IP 是 `192.168.1.25`，手机网址就是 `http://192.168.1.25:3000`。必须保留末尾的 `:3000`。

手机、电脑和 Switch 必须连接同一个普通 Wi-Fi。访客网络、VPN、移动数据或路由器的设备隔离功能可能导致无法连接。

## 五、连接 Switch

1. 在 Switch 的“设置 → 互联网 → 互联网设置 → 当前网络”中查看 Switch IP。
2. 启动旷野之息并进入实际存档，不要停留在标题画面。
3. 打开 Tesla 菜单，启动 Hyrule Bridge Ultra。
4. 网页右上角输入当前 Switch IP，然后点击“连接”。
5. 显示“坐标同步中”后即可使用。

发布包不会预填任何人的 Switch IP。网页会在当前浏览器保存使用者自己输入的地址。

## 六、初始之塔校准

1. 电脑端点击底部“初始塔模拟”，或按键盘 `T`。
2. 地图会显示初始之塔的标准模拟位置。
3. 模拟箭头应与初始之塔标记重合。
4. 再次点击按钮或再按一次 `T` 退出模拟。
5. 退出后连接 Switch，网页恢复显示真实位置。

该模拟只影响网页显示，不会读写游戏存档。

## 七、地图与导航

- 点击左侧分类可筛选神庙、塔、村落、装备、呀哈哈、任务和敌人等位置。
- 点击地图标记后选择“导航到这里”。
- 网页显示直线距离、目标方向和相对角色朝向的转向提示。
- 可切换中文或日语语音，并可随时关闭语音。
- `L` 定位林克，`Y` 打开筛选，`X` 切换足迹，`T` 切换初始塔模拟。
- 缩放按钮位于左下角；电脑也可使用鼠标滚轮。

导航是实时直线方向提示，不会自动计算道路、山路、河流或攀爬路线。

## 八、正常关闭

1. 网页右上角点击“断开”。
2. 关闭 Tesla 覆盖菜单中的插件。
3. 关闭游戏。
4. 回到电脑启动窗口，按 `Control+C` 停止网站服务。

插件带有断链自动停止保护，但仍建议按以上顺序关闭。

## 九、常见问题

### 手机打不开网站

- 确认网址包含 `http://` 和 `:3000`。
- 确认手机与电脑在同一 Wi-Fi。
- 关闭 VPN 和移动数据后重试。
- 检查电脑防火墙是否允许 Node.js 接收局域网连接。

### 网页显示“主机未连接”

- Switch IP 可能变化，请重新查看并输入。
- 确认插件已经启动。
- 确认三台设备没有连接访客网络。

### 显示“请启动旷野之息并进入存档”

必须进入可以控制林克的游戏存档；标题画面、读取画面和其他游戏不会提供坐标。

### Tesla 菜单没有插件

确认文件路径是 `SD:/switch/.overlays/HyruleBridgeUltra.ovl`，并确认没有多套嵌套文件夹或重复旧版文件。

### 关机异常

先断开网页并关闭插件。若仍有异常，移除该 `.ovl` 后确认系统恢复，再检查是否混装旧版本或其他冲突插件。
